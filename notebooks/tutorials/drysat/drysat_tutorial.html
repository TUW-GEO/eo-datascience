Soilmoisture analysis and validation for Mozambique

The aim of this notebook is to show a possible workflow for analyzing and validating soil moisture data in Mozambique retrieved from ground measurements and ASCAT satellite data. Due to the given ground measurement data we focus on 5 different areas Buzi, Chokw, Mabalane, Mabote and Muanza.

Workflow

  1. Obtaining ground data (access, visiualize(cardtopy, histogram)... )
  2. Obtaining ASCAT satellite data (access, transform data, df wide to lng, visiualize (histogram, spatial image)... )
  3. Visual validation (compare histograms)
  4. Statistic validation
